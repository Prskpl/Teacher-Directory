Teacher Directory App
A simple full-stack application using Flutter (frontend), Flask (backend), and MongoDB (database) to manage and display a list of teachers.

Tech Stack:
Frontend: Flutter

Backend: Flask (Python)

Database: MongoDB

API Communication: HTTP (REST)
